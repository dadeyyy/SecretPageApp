# SecretPageApp

## Visit here:
https://secret-page-app-one.vercel.app/login

## Features:
- Login, Signup, Signout using Supabase Auth
- Account Deletion
- Creating a secret
- Sending & Accepting Friend Request
- Viewing the secret of a Friend
